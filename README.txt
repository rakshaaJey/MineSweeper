# Minesweeper 

## Commands 
- -bomb x -> Overrides the number of bombs on the board
- -size r c -> Overrides the number of rows and columns on the board

## Terminal Commands
- r x y -> "Presses" the tile located at x y
- f x y -> Places a flag at the tile located at x y
- rmf x y -> Removes the flag located at x y
